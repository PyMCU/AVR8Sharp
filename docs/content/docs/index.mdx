## Introduction

Avr8Sharp is a .NET library to emulate AVR microcontrollers. It is designed to be used in
.NET applications, including web applications, desktop applications, and mobile applications.

It provides a simple and easy-to-use API for emulating AVR microcontrollers, allowing developers to
write and test AVR code without the need for physical hardware.

The library is written in C# and is compatible with .NET Core and .NET Framework.
